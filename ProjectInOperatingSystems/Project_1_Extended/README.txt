To run my project 1:

- Navigate via the command line to my Project_1_Extended folder
- Inside the folder will be:
    - a source code folder named source
    - a folder including the files in the jar named jar_files
    - a proj1.jar
    - the provided input.txt
    - this README.txt
- From inside the Project_1_Extended folder, run the command: java -jar proj1.jar
- An output.txt file will be produced in the Project_1_Extended folder
